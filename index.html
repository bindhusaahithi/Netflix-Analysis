Netflix Data Analysis

A data analysis project exploring Netflix Movies and TV Shows to uncover trends in content growth, genre popularity, country contribution, ratings, and director activity.

View GitHub Repository View Dataset

Project Overview

This project analyzes the Netflix Movies and TV Shows dataset to uncover trends in content growth, genre popularity, country contribution, and rating distribution.

Using Python, Pandas, and Seaborn, the project performs exploratory data analysis, data cleaning, and visualization to identify patterns in Netflix’s content strategy and understand how the platform’s catalog has evolved over time.

This project demonstrates practical skills in data cleaning, exploratory data analysis, visualization, and insight generation using a real-world dataset.

Tools & Technologies

  • 🐍 Python
  • 📊 Pandas
  • 🔢 NumPy
  • 📈 Matplotlib
  • 🎨 Seaborn
  • 📓 Jupyter Notebook

Dataset

Dataset: Netflix Movies and TV Shows Dataset

Size: ~8800+ titles

Features: Title, Type, Director, Country, Cast, Genre, Rating, Release Year

Source: Netflix Movies and TV Shows Dataset on Kaggle

Project Objectives

  • Analyze the growth of Netflix content over the years
  • Compare the distribution of Movies and TV Shows
  • Identify the most popular genres on Netflix
  • Find the top countries contributing content
  • Explore directors with the most titles
  • Examine the most common ratings across Netflix titles

Skills Demonstrated

  • Data Cleaning and Preprocessing
  • Exploratory Data Analysis (EDA)
  • Data Visualization
  • Trend Analysis
  • Insight Generation
  • Python Data Analysis Workflow

Project Workflow

  1. Data Collection: Obtained the Netflix dataset from Kaggle.
  2. Data Cleaning: Handled missing values and standardized relevant columns.
  3. Exploratory Data Analysis: Explored distribution across genres, countries, ratings, and content type.
  4. Data Visualization: Built visualizations using Matplotlib and Seaborn.
  5. Insight Extraction: Identified key trends in Netflix content growth and catalog structure.

Key Questions Answered

  • How has Netflix content grown over time?
  • Does Netflix have more Movies or TV Shows?
  • Which countries contribute the most content?
  • Which genres are most common on Netflix?
  • Which directors have the highest number of titles?
  • What ratings appear most frequently in the dataset?

Visual Analysis

Content Growth

Netflix experienced rapid content expansion between 2016 and 2019, with a peak around 2019. After this period, growth slightly declined.

Movies vs TV Shows

Movies dominate Netflix’s catalog compared to TV Shows, showing the platform still relies heavily on movie content while growing its series library.

Top Countries

The United States contributes the largest share of Netflix content, followed by India and other major markets, reflecting Netflix’s global distribution strategy.

Top Directors

Director analysis highlights the creators with the strongest presence in the dataset and helps identify frequent contributors to the Netflix catalog.

Top Genres

Genre distribution shows which content categories dominate the platform and reveals Netflix’s concentration in a few highly represented genres.

Top Ratings

Ratings analysis helps explain how Netflix content is distributed across audience groups and what type of maturity levels appear most often.

Key Insights

  • Netflix content expanded rapidly in recent years before a slight decline.
  • Movies make up a larger share of the Netflix catalog than TV Shows.
  • A few countries contribute a major portion of the total content available.
  • Some genres dominate the platform, showing strong content concentration in specific categories.
  • Ratings analysis provides a clear view of the target audience across Netflix titles.
  • Director analysis highlights recurring creators with a strong catalog presence.

Why This Project Matters

Conclusion

The analysis shows that Netflix rapidly expanded its content library between 2016 and 2019, with movies dominating the platform’s catalog. International content also plays an important role, reflecting Netflix’s strategy to reach global audiences.

These insights demonstrate how data analysis can help understand content trends and platform strategy in the streaming industry.
