Netflix Data Analysis

A data analysis project exploring Netflix Movies and TV Shows to uncover trends in content growth, genre popularity, country contribution, ratings, and director activity.

View GitHub Repository View Dataset

Project Tags

Business Problem

Streaming platforms manage massive content libraries across multiple genres, countries, and audience categories. Understanding how content is distributed over time helps uncover patterns in growth, popularity, and regional contribution. This project analyzes Netflix’s catalog to identify trends in content expansion, genre concentration, ratings, and country-wise production.

Project Overview

This project analyzes the Netflix Movies and TV Shows dataset to uncover trends in content growth, genre popularity, country contribution, and ratings. Using Python, Pandas, and visualization libraries, the analysis explores meaningful insights from Netflix’s global content catalog.
