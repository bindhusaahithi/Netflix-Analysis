Netflix Data Analysis

A data analysis project exploring Netflix Movies and TV Shows to uncover trends in content growth, genre popularity, country contribution, ratings, and director activity.

View GitHub Repository View Dataset

Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python and popular data analysis libraries. The main goal is to understand how Netflix content has grown over time, compare Movies and TV Shows, identify the most common genres, find the top contributing countries, examine common audience ratings, and explore directors with the highest number of titles.

This project demonstrates practical skills in data cleaning, exploratory data analysis, visualization, and insight generation using a real-world dataset.

Tools & Technologies

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
  • Jupyter Notebook

Dataset

This project uses the Netflix Movies and TV Shows dataset, which includes information such as title, type, country, director, cast, rating, release year, and genre.

Dataset Source: Netflix Movies and TV Shows Dataset on Kaggle

Project Objectives

  • Analyze the growth of Netflix content over the years
  • Compare the distribution of Movies and TV Shows
  • Identify the most popular genres on Netflix
  • Find the top countries contributing content
  • Explore directors with the most titles
  • Examine the most common ratings across Netflix titles

Key Questions Answered

  • How has Netflix content grown over time?
  • Does Netflix have more Movies or TV Shows?
  • Which countries contribute the most content?
  • Which genres are most common on Netflix?
  • Which directors have the highest number of titles?
  • What ratings appear most frequently in the dataset?

Visual Analysis

Content Growth

This visualization shows how Netflix content expanded rapidly over the years, highlighting a strong increase before a slight decline.

Movies vs TV Shows

This chart compares the number of Movies and TV Shows available on Netflix and shows that Movies form the larger share of the platform.

Top Countries

This analysis highlights the countries contributing the most titles to Netflix, showing a strong concentration from a few key regions.

Top Directors

This chart identifies the directors with the highest number of Netflix titles, helping reveal the most frequent content creators in the dataset.

Top Genres

This visualization shows the most common genres available on Netflix and highlights the kinds of content that dominate the catalog.

Top Ratings

This analysis displays the most frequent content ratings in the dataset and helps explain how Netflix content is distributed across audience groups.

Key Insights

  • Netflix content experienced rapid growth in recent years before a slight decline.
  • Movies make up a larger portion of the Netflix catalog than TV Shows.
  • A few countries contribute a major share of the total content available.
  • Some genres dominate the platform, showing strong content concentration in specific categories.
  • Ratings analysis provides a clearer view of the target audience across Netflix titles.
  • Director analysis helps identify the creators with the strongest presence in the dataset.
