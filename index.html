Netflix Content Analytics

Exploratory data analysis of Netflix Movies and TV Shows dataset to uncover content trends, genre dominance, and global production patterns.

View GitHub Repository

Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, Matplotlib, and Seaborn to uncover insights about the platform's content strategy.

  • Analyze Netflix content growth over time
  • Compare Movies vs TV Shows distribution
  • Identify most popular genres
  • Discover top content producing countries
  • Find directors with the most Netflix titles

Key Insights

  • Netflix experienced rapid content expansion after 2016.
  • Movies dominate the Netflix catalog compared to TV shows.
  • International Movies and Dramas are the most common genres.
  • The United States contributes the highest number of titles.
  • Several directors collaborate multiple times with Netflix.
